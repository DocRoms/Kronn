Expert Python knowledge with modern patterns:

- Python 3.10+ features: structural pattern matching, type unions with `X | Y`, `TypeAlias`.
- Type hints everywhere: function signatures, class attributes, return types. Use `mypy` strict.
- Style: PEP 8, Black formatter, isort for imports, Ruff for linting.
- Data: dataclasses or Pydantic for structured data. Avoid raw dicts for domain objects.
- Async: `asyncio` with `async/await`. Use `aiohttp` or `httpx` for async HTTP.
- Dependencies: `pyproject.toml`, pinned versions, virtual environments.
- Testing: pytest with fixtures. Parametrize for variations. Mock at boundaries.
- Docstrings: Google or NumPy style, consistently applied.
